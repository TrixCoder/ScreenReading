# ScreenReading

<p>A project by <a href="https://github.com/TrixCoder">TrixCoder</a> for screen reading using TensorFlow, OpenCV, and MSS.</p>
<h2>Installation</h2>
<p>To install the dependencies, run:</p>
<pre>
    <code>
    pip install -r requirements.txt
    </code>
</pre>
